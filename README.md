# Photo Style Lab

![Photo Style Lab hero](./public/hero.png)

사진을 열고 여러 스타일 프리셋을 적용해 결과를 비교한 뒤 이미지로 저장하는 Remix 기반 웹 앱입니다.

## 기능

- 이미지 업로드와 드래그 앤 드롭
- 원본/변환 이미지 비교 보기
- 빈티지, 흑백, 세피아, 고대비, 시네마틱 등 스타일 프리셋
- Canvas 기반 이미지 변환
- 변환 결과 다운로드

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run typecheck
npm run lint
npm run build
npm start
```

## 기술 스택

- Remix
- React
- TypeScript
- Tailwind CSS
- Canvas API
